# Hackintosh-Deskmini310

# 我的配置

- 主板: ASROCK 310-com
- 硬盘: Fanxiang S500PRO 1TB
- CPU: i7-8700
- 无线网卡: BCM94360CS2+转接卡
- 内存: 16G DDR4 X2

# 功能

- 睡眠唤醒: 正常
- 蓝牙: 正常
- Wi-Fi: 正常
- 有线网卡: 正常
- 声卡: 正常
- CPU 变频: 正常
- Hand Off 接力: OK/正常

# 使用

## Ventura

- 先更改 OpenCore 配置文件中的三码
- 安装系统

## Sonoma

- 先更改 OpenCore 配置文件中的三码
- 安装系统
- 进系统后需要安装 OpenCore-Patcher，Wi-Fi 才能正常
  - 下载 OpenCore-Patcher
  - 点击 “Post-Install Root Patch” 按钮
  - 按提示安装后重启
